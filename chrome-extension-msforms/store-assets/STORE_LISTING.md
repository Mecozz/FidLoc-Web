# Chrome Web Store Listing Information

## Extension Name
FidLoc Forms Filler

## Short Description (132 characters max)
Auto-fill Microsoft Forms with equipment serial numbers. Built for Fidium Fiber technicians to report missing/installed CPE inventory.

## Detailed Description
FidLoc Forms Filler streamlines the process of submitting equipment inventory reports to Microsoft Forms.

**Perfect for fiber technicians who need to:**
• Report multiple missing or installed serial numbers
• Submit equipment reconciliation data quickly
• Avoid repetitive manual form entry

**How it works:**
1. Paste your list of serial numbers (one per line)
2. Set your default values (name, garage, status)
3. Click "Fill Next & Submit" to auto-fill the form
4. Submit and repeat for the next serial number

**Features:**
• Queue multiple serial numbers for batch processing
• Saves your default technician name and garage
• Auto-fills all form fields including radio buttons
• Progress tracking shows remaining items
• Skip button for problematic entries
• Works with Microsoft Forms

**Built for Fidium Fiber / Consolidated Communications technicians**, but can be adapted for any MS Forms workflow that requires repetitive data entry.

## Privacy Policy
This extension:
- Only activates on forms.office.com pages
- Stores your preferences locally in Chrome (name, garage, status)
- Does NOT collect or transmit any personal data
- Does NOT require sign-in
- All data stays on your device

No data is sent to external servers. The extension only interacts with Microsoft Forms pages to fill in form fields.
